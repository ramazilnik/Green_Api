# Клиент для интеграции с GREEN-API

Данный проект представляет собой одностраничное веб-приложение для взаимодействия с методами сервиса GREEN-API. Проект выполнен в рамках тестового задания на позицию Девопс разработчик.

* Видео-демонстрация работы: https://drive.google.com/file/d/1AUuH67LqHzE6qSUN2iWUfVpBa-KUe8Lf/view?usp=drive_link

## Технологический стек
* Frontend: HTML5, JavaScript (ES6, Fetch API)
* Стилизация: Tailwind CSS (CDN)
* Хостинг и деплой: GitHub Pages

## Реализованный функционал
Интерфейс позволяет ввести параметры подключения к инстансу (idInstance, apiTokenInstance) и протестировать работу следующих методов:

1. getSettings — получение настроек текущего инстанса.
2. getStateInstance — проверка состояния авторизации инстанса.
3. sendMessage — отправка текстового сообщения на указанный номер телефона.
4. sendFileByUrl — отправка файла посредством прямой ссылки.

Результаты выполнения всех запросов выводятся в формате JSON в отдельное текстовое поле, доступное только для чтения.
